# arikmukherjee.github.io

A modern, responsive personal link aggregator and portfolio landing page hosted on GitHub Pages.

## Overview

This project serves as a centralized hub for sharing personal links and social profiles. It features a clean, modern design with a gradient background, animated visual effects, and dark/light mode support for an enhanced user experience.

## Features

- **Responsive Design**: Fully responsive layout that works beautifully on desktop, tablet, and mobile devices
- **Dark Mode**: Toggle between light and dark themes with smooth transitions
- **Animated Background**: Dynamic animated bubble effects that add visual interest to the page
- **Modern UI**: Clean, minimalist design using a card-based layout
- **Link Aggregation**: Centralized landing page to direct visitors to various profiles and projects

## Tech Stack

- HTML5
- CSS3 (with CSS variables for theming)
- JavaScript (for dark mode toggle and interactivity)

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. The site will be live at `https://arikmukherjee.github.io`

## Hosting on GitHub Pages

This project is hosted on **GitHub Pages**, GitHub's built-in static site hosting service. 

### How It Works

1. **Repository Setup**: The repository is named `arikmukherjee.github.io` following GitHub Pages naming convention
2. **Main Branch Deployment**: Any files pushed to the `main` branch are automatically deployed
3. **Automatic Publishing**: GitHub Pages automatically builds and publishes the site within seconds of updates

### Deployment Process

1. Make changes to your files locally (HTML, CSS, JavaScript)
2. Commit your changes: `git commit -m "Update description"`
3. Push to the main branch: `git push origin main`
4. GitHub Pages automatically deploys within seconds
5. Access your site at `https://arikmukherjee.github.io`

### Static Site Benefits

- **Free hosting** with GitHub
- **HTTPS enabled** by default
- **No build process required** - static files are served directly
- **Version control** built-in with Git
- **Instant updates** - changes are live immediately after pushing

## Customization

The design uses CSS custom properties (variables) for easy theming. Modify the color scheme by updating the `--primary`, `--bg`, and `--text` variables in the `:root` selector.
